Uso básico de la línea de comandos

Qué es la línea de comandos ó shell ó CMD (Command)

https://es.wikipedia.org/wiki/Interfaz_de_l%C3%ADnea_de_comandos

  • Es una interfaz de usuario, un programa que recibe órdenes (comandos) para otros programas o para el sistema operativo.
  • El CMD es uno de muchos tipos o sabores de líneas de comando o shell.

Cómo le decimos a la computadora que haga una u otra cosa

  • Por medio de oraciones con cierta sintaxis, a través de la Línea de Comandos.
  • La sintaxis[[20230706172210]] son las reglas para la construcción de oraciones, es decir, cómo combinar sujeto, verbo y predicado.

Podría ser algo así:

  • Verbos: comandos
  • Sujeto: archivos
  • Predicado: ubicaciones, parámetros, particularidades.

Qué es un comando

Los comandos o instrucciones son pequeños programas incorporados al sistema operativo o partes de otros programas.

Dónde están los comandos.

Generalmente están en capetas las carpetas (/bin, /usr/bin)

Prompt

El prompt es el que indica que el shell está esperando espera de órdenes. Varía dependiendo del interprete de comandos.

Por ejemplo, en el CMD, el prompt es:

C:\ruta del directorio actual\>

En la imagen aparece el prompt compuesto por:

  • Identificador de disco local
  • ruta del directorio actual
  • símbolo de mayor que
  • cursor (ese cuadro parpadeante en blanco)

Práctica: Lista comandos del CMD usaremos hoy

Ver: https://www.xataka.com/basics/comandos-basicos-para-dar-tus-primeros-pasos-consola-windows-cmd

Probar escribir los siguientes comandos u órdenes en el seguida del prompt:

cls - Borrar pantalla:

chdir - Muestra la ruta del directorio en el que estamos:

dir - Listar el contenido de un directorio:

Nota: Observar que las primeras líneas de la lista son los directorios . y .. ¿Qué son?

HERENCIA DE ARCHIVOS

  • . -> representa el directorio donde estoy, el actual -> “sig02”
  • .. -> representa el directoria padre -> “Usuarios”

Ver navegador de archivos:

cd - Cambia de directorio:

Ejemplo: cambiar al directorio padre: cd ..

Nota: Poner atención en que hay un espacio entre el comando cd y los dos puntos " .. "

mkdir - crear un directorio:

Crear un directorio:
mkdir nubes_de_puntos

copy - copiar un archivo:

Crear un directorio:
copy nube1.txt nube1_copia.txt Tener atención en que hay un espacio entre el comando cd y los dos puntos ...

del - borrar un archivo:

Crear un directorio:
del nube1_copia.txt Tener atención en que hay un espacio entre el comando cd y los dos puntos ...

Otros comandos:

mv -> mover
